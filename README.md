# AI Voice Assistant

A real-time, multi-user AI Voice Assistant built with Python, Flask, and the Google Gemini API.

## Features
- Push-to-talk voice recognition
- Multi-user support with private chat history
- AI-powered responses using Gemini Flash
- Clean, responsive UI with dark mode logic
